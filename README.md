# بوت استخراج الأسئلة إلى قاعدة بيانات

## أوامر واجهة التحكم (تعديل بعد الإدخال)

بعد ما تبعث ملف وتتعبى القاعدة، هاي الأوامر لتعديل أي سؤال لاحقاً:

- `/edit كلمة_من_السؤال` — بحث نصي (بيطلعلك أزرار بالأسئلة المطابقة)
- `/browse` — تصفح: مادة ← شيت ← قائمة أسئلة
- `/goto 42` — وصول مباشر لو بتعرف رقم السؤال

بعد ما تختار سؤال، بتظهرلك 4 أزرار بس:
- 📝 نص السؤال
- ✅ الإجابات (تعديل نص أي خيار، أو تغيير الإجابة الصحيحة)
- 🏷️ التصنيف (حذف/إضافة، وبتمر بنفس فحص التشابه الذكي)
- 📌 الملاحظة (شرح + انتبه)

كل شي تاني (isCorrect، SubjectTag، TagStatistic) بيتحدث أوتوماتيك بدون ما تلمسه.

## التشغيل
```
pip install -r requirements.txt
python bot.py
```
بيتولد ملف `database.db` بجانب الكود أول ما تشغله، ويضل هو نفسه دايماً
(كل ملف txt بتبعته بيضاف فوق الموجود، ما بينمسح شي).

⚠️ التوكن موجود بملف `.env` جاهز للتجربة. لما تخلص تجربة، بنصحك ترجع
لـ BotFather وتعمل `/revoke` وتولد توكن جديد إذا بدك تستمر فيه بشكل دائم،
لأنو هاد التوكن صار موجود بسجل هاد المحادثة.

## شو مبني وشو لسا لازم تدقيق فيه

✅ **جاهز ومجرّب فعلياً** (اختبرته بملف الخمجية وطلع 13 شيت / 99 سؤال / 478 إجابة):
- الـ Parser (استخراج الشيتات/الأسئلة/الخيارات/الشرح/انتبه)
- دمج شرح + انتبه بعمود note
- المطابقة الذكية للتصنيفات (تطابق تام / تشابه مشكوك بيسأل / جديد)
- توليد uuid تلقائي بدون تكرار (معتمد على id الفعلي بقاعدة SQLite)
- تصنيف مزدوج (عريض + محدد) لكل سؤال

⚠️ **لازم تتأكد/تعدّل قبل ما تعتبره جاهز فعلياً**:
1. **قائمة Subject** (بملف `db.py`, متغير `SUBJECTS`) — حالياً فيها مادة وحدة تجريبية بس
   ("الأمراض الخمجية"). زوّد باقي المواد بنفس الصيغة `(uuid, name, term_uuid)`.
2. **السنة المزدوجة** (متل "2022/2021") — حالياً بآخد أول رقم بس (2022) وبتجاهل التاني.
   إذا هاد مو الصح، قلي شو المنطق الصحيح.
3. **السؤال "سؤال تابع للداخليات الصغيرة"** طلع أثناء الفحص كـ"تصنيف جديد" —
   هاد مش تصنيف طبي حقيقي، ظاهر إنه ملاحظة داخل الملف مو عنوان تصنيف. لازم
   تراجعه يدوياً بالملف أو تقولي كيف نتعامل معه.
4. **البوت لازم يضل شغال عندك بشكل مستمر** (`python bot.py`) وقت ما بدك تستخدمه —
   إذا سكرت الطرفية (terminal) بينوقف. لاحقاً إذا حبيت يشتغل ٢٤/٧ محتاج تحطه
   عسيرفر (VPS) أو خدمة استضافة.
